## Neuladen eines Roster
:::cmd
/roster reload **roster**
:::

Es kann vorkommen, dass ein Roster mal nicht zu 100% aktuell ist. In diesem Fall, kann mit dem Befehl ein neuladen des
Roster veranlasst werden.
